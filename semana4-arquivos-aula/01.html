HTML

Quando uma página é carregada, o navegador o Document Object Model (DOM)dela. A partir do DOM é possível:

  • Acessar e alterar o conteúdo de elementos existentes na página;
  • Remover e adicionar elementos na página;
  • Acessar, alterar e adicionar estilo (CSS) aos elementos na página;
  • Manipular eventos relacionados aos elementos na página.

Os elementos HTML também podem ser acessados por meio métodos e objetos do DOM.

Os elementos HTML podem ter suas propriedades e atributos manipulados.
